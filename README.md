# Portfolio Website

This is my personal portfolio website built with React, Vite, and Tailwind CSS. It showcases my skills, projects, and provides a way for potential employers or clients to contact me.

## Features

- Responsive design
- Interactive 3D element in the hero section
- Skills showcase
- Project portfolio
- Contact form using Formspree

## Technologies Used

- React
- TypeScript
- Vite
- Tailwind CSS
- Three.js (via @react-three/fiber and @react-three/drei)
- Lucide React for icons
- Formspree for contact form submissions

## Getting Started

1. Clone the repository
2. Install dependencies with `npm install`
3. Run the development server with `npm run dev`
4. Build for production with `npm run build`

## Deployment

This site is deployed on Netlify. Any pushes to the main branch will trigger a new deployment.
